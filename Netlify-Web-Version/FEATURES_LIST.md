# Tooth Kingdom Adventure - Frontend Feature List

This document provides a comprehensive breakdown of all identified features currently implemented or wireframed in the Tooth Kingdom Adventure frontend.

## 1. Authentication & Security 🔐
*   **Dual-Mode Access**: Toggle between **Login** and **Sign Up** within the same interface.
*   **Hero Profile Creation**: Captures "Hero Name" during registration.
*   **Social Sign-On (SSO)**:
    *   Continue with **Google**
    *   Continue with **Facebook**
    *   Continue with **Apple**
*   **AI-Powered Authentication** (Smart Login):
    *   **Face ID**: Instant facial recognition login.
    *   **Touch ID**: Biometric fingerprint authentication.
    *   **Voice ID**: Secure voice pattern recognition.
*   **Multi-Factor Security (2FA)**:
    *   Toggle for OTP protection.
    *   **OTP Verification Screen**: Dedicated UI for entering verification codes.
*   **Password Management**:
    *   Show/Hide password toggle.
    *   Forgot password recovery path.

## 2. Core Gameplay & Adventures 🎮
*   **Adventure Hub (Dashboard)**: The main control center showing level, stars, and current streak.
*   **Chapter Progression Map**: Visual journey through different dental health "Chapters".
*   **Custom Adventure Slots**: Built-in 16:9 placeholders for future user-created games, ensuring easy UI expansion.
*   **Mini-Games**:
    *   **Brushing Game**: Interactive brushing simulation.
    *   **Flossing Game**: Tutorial-style game for flossing technique.
    *   **Food Sorting Game**: Educational game for healthy vs. sugary foods.
    *   **Germ Attack Game**: Arcade-style game where heroes battle cavity-causing germs.
    *   **Quiz Game**: Knowledge tests on dental hygiene.
*   **Game Engine**: Backend logic for handling game states, collisions, and scoring.

## 3. Educational Ecosystem 📚
*   **Learning Academy**:
    *   Courses led by an **AI Tutor**.
    *   Offline access for learning anywhere.
*   **Learning Resources**:
    *   Video library.
    *   PDF guides and downloads.
    *   Interactive educational lessons.
*   **Daily Brushing Lessons**: Step-by-step guided instructions for daily hygiene routines.

## 4. Progress Tracking & Gamification 🏆
*   **Streak System**: Tracks consecutive days of brushing with milestone rewards (e.g., 14-day streak).
*   **Live Progress Overview**: Real-time tracking of Morning vs. Evening brushing tasks.
*   **User Statistics**: Detailed analysis of time spent, level progress, and total stars earned.
*   **Achievement System**: Unlockable badges for completing various dental milestones.
*   **Rewards Cabinet**: A collection of gifts and unlocked items earned through progress.
*   **Activity Calendar**: A visual history (calendar view) of all brushing sessions and activities.
*   **Victory Flow**: 
    *   **Lesson Complete**: Festive success screen after finishing a module.
    *   **Reward Unlocked**: Special celebratory screen for new item acquisition.

## 5. User Interface & Personalized Experience 🎨
*   **Hero Selection**: Choice of different characters to represent the user in the kingdom.
*   **Dynamic Avatar**: Animated avatars that update based on selection.
*   **Personalized AI Assistant**:
    *   **Daily AI Tips**: Smart suggestions based on brushing habits.
    *   **AI Chat Assistant**: Conversational interface for dental health questions.
    *   **Goal Predictions**: AI forecasts success based on current performance.
*   **Notification System**:
    *   Custom bell notifications for reminders and updates.
    *   Dedicated notifications management screen.
*   **Adaptive Theme**: Toggle between **Light Mode** and **Dark Mode**.

## 6. App Settings & Customization ⚙️
*   **Audio Controls**: Toggle for sound effects and music.
*   **Language Support**: Multi-language localization settings (English pre-built).
*   **Privacy & Support**:
    *   Dedicated Privacy settings.
    *   Help & FAQ support center.
    *   About section and version tracking.
*   **Session Management**: Secure sign-out functionality.
