# 🧶 Chapter 03: Flossing Adventures

### 🌟 Technical Overview
- **Title:** Flossing Adventures
- **Status:** Integrated
- **Game Type:** `flossing-fantasy`
- **Primary Component:** `src/components/games/RoyalRopeRescue.tsx`

---

### 🎨 Frontend Details
- **Visuals:** Uses `assetColoringBook` for the map icon.
- **Animation:** `framer-motion` handles the rope elasticity and "Rescue" popups.
- **Key Props:** `onComplete`, `onExit`.

---

### 🧠 Core Game Logic (Blueprint: `RoyalRopeRescueLogic.cs`)
- **Mechanic:** Physics-based Obstacle Navigation (Runner).
- **Physics Constant:** 
    - `gravity`: -9.8
    - `flapStrength`: 5.0
- **Logic:**
    - Player launches a "floss line" to swing between gaps.
    - **Gap Scoring:** 20 points for pass, 30 points for a "Perfect" center-line pass.
    - **Lives:** 3 Hearts (Lives) allowed before Game Over.
- **Invincibility:** 0.9s of "i-frames" after taking damage.

---

### 🔄 Data & Sync Flow
1. **Update:** `updateUserData` is called with the total stars earned (max 15).
2. **XP:** Awards **30 XP** per successful gap pass.
3. **Backend:** SQLite `chapter_progress` table stores the `stars_earned` for this specific `chapter_id: 3`.
