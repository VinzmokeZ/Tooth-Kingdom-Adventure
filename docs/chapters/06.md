# 🧨 Chapter 06: Battle Dentist

### 🌟 Technical Overview
- **Title:** Battle Dentist
- **Status:** Retro Action Challenge
- **Game Type:** `external`
- **External URL:** `/games/beatrix-dentist/index.html`
- **Primary Component:** `src/components/games/ExternalGameWrapper.tsx`

---

### 🎨 Frontend Details
- **Style:** 8-bit retro aesthetic.
- **Integration:** Uses the `GameEngine` to bridge the retro JS game with the modern React RPG system.
- **Controls:** Virtual Joystick (on mobile) or WASD/Arrow Keys (on Desktop).

---

### 🧠 Core Game Logic (Blueprint: `BattleDentistLogic.gml`)
- **Mechanic:** Top-Down Combat (Cleaning Blast).
- **Physics:** `player_speed = 5`.
- **Logic:**
    - **Cleaning Blast:** A circular AOE (Area of Effect) attack that removes Grime Monsters.
    - **Monsters:** They spawn at the edges and move toward the "Crown Tooth" in the center.
- **Score:** 100 points per Grime Monster destroyed.

---

### 🔄 Data & Sync Flow
1. **Sync:** Post-game telemetry is sent to `/game/{uid}/session-log`.
2. **Persistence:** The `Total Monsters Defeated` count is updated in the SQLite `character_stats` table.
3. **Rewards:** Awards the "Battle-Hardened" badge if Chapter 6 is completed without losing a life.
