# src_complete_inventory.md: Folder Breakdown

- **`src/main.tsx`**: Entry point + Desktop Scroll Helper.
- **`src/App.tsx`**: Root component + State initialization.
- **`src/components/screens/`**: All 27 pages of the app (Dashboard, Parent, Chapters, etc.).
- **`src/components/games/`**: The 6 chapter game components and the `GameEngine`.
- **`src/components/ui/`**: 48+ reusable components (Buttons, Cards, Modals).
- **`src/context/`**: Auth and Game logic providers.
- **`src/services/rpgService.ts`**: The "Heart" of the RPG engine.
- **`src/lib/firebase.ts`**: API and Firebase configuration center.
- **`src/data/`**: Static definitions for Chapters, Quests, and Rewards.
- **`src/assets/`**: Images, character SVGs, and food items.
